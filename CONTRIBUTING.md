## Ways to Contribute
- **Code**: Fix bugs, add features, optimize performance
- **Docs**: Improve README, add comments, create tutorials
- **Testing**: Write tests, report bugs, suggest improvements
- **Reviews**: Review PRs and provide feedback

## Development Setup
### Prerequisites
- C++17 compiler (MSVC, GCC, Clang)
- Git

### Quick Start
1. Fork and clone the repo
2. Build: `.\compile.bat` (Windows)
3. Download model files (weights, config, vocab)
4. Run: `.\llm_engine.exe`

## Guidelines
- Use feature branches: `git checkout -b feat/your-feature`
- Follow conventional commits: `feat:`, `fix:`, `docs:`, etc.
- Test your changes thoroughly
- Keep PRs focused and well-described

## Code Standards
- C++17 with modern best practices
- 4-space indentation, <100 char lines
- Meaningful names and comments
- Include tests for new features

See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) for community guidelines.
